# Relevant organizations for RS

| Organization                                                               | Description                                                                                                         |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [eScience Center](https://www.esciencecenter.nl/)                          | A Dutch organization that supports researchers in developing and using software to enhance their research.          |
| [NL-RSE](https://nl-rse.org/)                                              | A Dutch community of RS Engineers (RSEs) that aims to professionalize the role of RSEs in the Netherlands.          |
| [Research Software Alliance (ReSA)](https://www.researchsoft.org/)         | An international organization that aims to bring together RS communities to collaborate on the advancement of RS.   |
| [Free Software Foundation (FSF)](https://www.fsf.org/)                     | An organization that promotes the use of free software and advocates for the rights of software users.              |
| [Open Source Initiative (OSI)](https://opensource.org/)                    | An organization that promotes and protects open source software.                                                    |
| [Free and Open Source Software (FOSS)](https://freeopensourcesoftware.org) | A community that promotes the use of free and open source software.                                                 |
| [ELIXIR](https://elixir-europe.org/)                                       | A European research infrastructure that supports life science research and provides tools.                          |
| [Software Sustainability Institute (SSI)](https://www.software.ac.uk/)     | A UK-based organization that supports researchers in developing sustainable software.                               |
| [Research Data Alliance (RDA)](https://www.rd-alliance.org/)               | An international organization that aims to build the social and technical bridges that enable open sharing of data. |
| [FORCE11](https://www.force11.org/)                                        | An international organization that aims to improve scholarly communication and e-scholarship.                       |
| [Software Heritage](https://www.softwareheritage.org/)                     | An organization that aims to collect, preserve, and share all publicly available software source code.              |
| [Code for Science & Society](https://codeforscience.org/)                  | An organization that supports open source projects in the research community.                                       |
| [Open Science NL](https://www.openscience.nl/)                             | A Dutch organization that promotes open science practices in the Netherlands.                                       |
| [The Carpentries](https://carpentries.org/)                                | An international community that teaches foundational coding and data science skills to researchers.                 |
| [Code Refinery](https://coderefinery.org/)                                 | An international project that provides training in RS development.                                                  |
| [DReaMS](https://dreams-community.github.io/)                              | DReaMS (Dutch Research and Medical Software) is a collaboration between DCCs of the Dutch UMCs, eScience Center, Elixir-NL, Health-RI, and TDCC-LSH. |
